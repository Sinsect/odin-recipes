Blueberry Banana Smoothie

Description

A smoothie loaded with delicious nutrients.

Highly customizable. Easy cleanup with immersion blender.

Ingredients

  • Frozen Blueberries
  • Banana
  • Milk or non-dairy beverage of choice
  • Oats
  • Honey if needing some extra sweetness
  • (Optional) Chia seeds, flax seed

Steps

  1. Add blueberries and Milk/substitute to blending container, blend thoroughly.
  2. Add banana, blend.
  3. Add oats, blend. Ensure all oats are submerged if using immersion blender.
  4. If using honey and/or seeds, add them here.
  5. Transfer to drinking vessel and consume.
